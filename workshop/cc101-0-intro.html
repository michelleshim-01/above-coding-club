Part 0

시작 전에

설치를 시작하기 전에, 오늘 우리가 다룰 도구들을 짧게 정리해요.

처음 보는 단어가 많아 보일 수 있지만, 워크샵에서는 같이 따라가니까 미리 외울 필요는 없어요. 머릿속에 큰 그림 하나만 그려두는 마음으로 가볍게 읽어보세요.

요즘의 코딩이 뭐예요?

예전 코딩은 영어 같은 프로그래밍 언어로 컴퓨터한테 일일이 지시하는 일이었어요. 지금은 달라요. 한국어로 "이런 걸 만들어줘"라고 말하면 AI가 알아듣고 코드를 써줘요. 우리는 무엇을 만들고 싶은지만 잘 말하면 돼요.

안전해요?

오늘 설치하는 건 전 세계 개발자들이 매일 쓰는 공식 프로그램이에요. VS Code · Claude Code · Git · Homebrew — 모두 출처가 분명한 도구예요.

비밀번호 입력은 정식 앱 설치 때와 같은 권한 확인 절차고, 모든 설치는 쉽게 지울 수 있어요.

Claude랑 Claude Code랑 뭐가 달라요?

같은 Anthropic의 같은 AI 모델인데, 만나는 창구가 달라요.

Claude는 카카오톡 상담사처럼 브라우저 안에서 대화만 해요. 질문에 답해주죠.
Claude Code는 직접 출장 온 전문가예요. 내 컴퓨터 안에 들어와서 파일도 만들고 명령도 실행해요.

ClaudeClaude Code
작동 위치웹 브라우저내 컴퓨터 터미널
할 수 있는 일대화, 질문 답변파일 생성/수정, 명령어 실행
비유카카오톡 상담사직접 출장 온 전문가

터미널이 뭐예요?

마우스로 클릭하지 않고, 키보드로 명령을 보내서 컴퓨터와 소통하는 창이에요. 평소 우리는 터치스크린 키오스크처럼 마우스로 메뉴를 눌러서 컴퓨터를 써요. 터미널은 직원에게 직접 말로 주문하는 거랑 비슷해요.

Claude Code를 쓸 때는 명령어를 외울 필요가 없어요. 한국어로 하고 싶은 말을 입력하면 돼요.

VS Code는 뭐예요?

코드를 보고 편집하는 작업 공간이에요. 글을 쓸 때 워드 프로세서가 있다면, 코드 작업엔 VS Code예요.

그리고 VS Code 안에는 터미널도 내장되어 있어서, 오늘은 VS Code 하나만 열면 모든 게 다 됩니다.
