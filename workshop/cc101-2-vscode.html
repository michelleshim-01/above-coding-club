VS Code 활용법

VS Code 필수 설정과 단축키를 익혀봅니다.

필수 설정

1

밝은 테마로 변경

눈에 편한 밝은 테마로 바꿔봅시다.

  • Mac: Code → Settings → Color Theme → Light Modern
  • Windows: File → Preferences → Color Theme → Light Modern
2

한국어 언어팩 설치 (선택)

Extensions (Cmd+Shift+X / Ctrl+Shift+X) → "Korean" 검색 → Korean Language Pack 설치 → VS Code 재시작

3

Auto Save 켜기

File → Auto Save 체크. 파일을 수정하면 자동으로 저장됩니다.

기본 조작법

  • 파일 탐색기 — 왼쪽 사이드바에서 폴더/파일 구조를 한눈에 볼 수 있습니다
  • 터미널 열기 — Ctrl+` (백틱) 또는 상단 메뉴 Terminal → New Terminal
  • 파일 검색 — Cmd+P (Mac) / Ctrl+P (Windows)
  • 전체 검색 — Cmd+Shift+F (Mac) / Ctrl+Shift+F (Windows)
